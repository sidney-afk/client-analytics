## What changed, and how to revert each piece

### 1. Dead code removed — `_calConflictNotified` (`index.html`)
A `Set` that was constructed and `.delete()`'d but **never `.add()`'d or `.has()`'d**
— always empty, never read, so its `.delete()` was a no-op. The conflict dialog it
once guarded no longer exists. Removed the declaration (+ comment) and the lone
`.delete(realId)` call together.
- **Revert:** `git checkout d83d1cd -- index.html` (restores the pre-cleanup file), or
  revert commit `de2dd25`.

### 2. Spent CI files deleted

— all 118 listed branches are gone from origin. Cannot affect the running app or any
cron.
